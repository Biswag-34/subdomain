# Nikoo Homes 8 Website Data Pack

Created for a high-conversion real estate landing/project page.

## What is inside

- `data/verified_project_data.json` — clean canonical project data.
- `data/unit_matrix_official_public_site.csv` — unit configuration matrix from the public Nikoo Homes 8 page.
- `data/indicative_pricing_cross_source.csv` — pricing bands cross-checked from public partner sources.
- `data/amenities_master_list.csv` — amenities grouped by storytelling priority.
- `data/landmarks_and_connectivity.csv` — location advantage data.
- `data/faqs.csv` — website FAQ data.
- `copy/website_copy_pack.md` — ready-to-use landing page copy.
- `strategy/website_architecture_and_lead_strategy.md` — CTA and page architecture plan.
- `assets/remote_asset_inventory.csv` — brochure, master plan, floor-plan and image URLs.
- `implementation/nikooHomes8.ts` — TypeScript data object for Next.js/React.
- `implementation/seo_schema_jsonld.json` — basic JSON-LD schema starter.
- `legal/disclaimer_notes.md` — safe publishing notes.

## Most important implementation rule

Build the page around “garden living”, keep phase-level facts clearly labelled, use a gated cost-sheet CTA for detailed pricing, and keep persistent WhatsApp/call/site-visit touchpoints.
